# api-party
#### Cara menggunakan
```
isi consumer_key, consumer_secret, access_token, access_token_secret, weather_api_key berdasarkan key yang anda miliki. kemudian isi query yang merupakan nama kota.
localhost:3000/search?q=<nama kota>
contoh:
localhost:3000/search?q=Yogyakarta
twitter anda akan otomatis memposting "Based on data 2017-02-01 18:00:00, there is will light rain on Yogyakarta"
```
